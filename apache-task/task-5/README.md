Є посилання:
site.com/aaaa/bbbb/cccc/.../kkkk, де 
/aaaa/bbbb/cccc/.../kkkk - шлях з будь якою кількістю вкладень

Якщо в запиті є файл або директорія яка починається з крапки (прихований файл або директорія) і цей файл/директорія існує, то віддати 301 redirect  на site.com/forbidden

Інакше, якщо файл не існує, то переписати посилання на:
site.com/file.php?arg=path
